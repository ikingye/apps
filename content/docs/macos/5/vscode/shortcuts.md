# VScode 快捷键

{{< tabs "vscode shortcut" >}}
{{< tab "MacOS" >}}

## 选择

- 选中所有同一个词 `cmd + shift + L`

## 跳转

- 左右括号跳转 `cmd + shift + \`
- 跳到定义 `cmd + 鼠标左键单击`
- 返回光标上一个位置 `ctrl + -`

## 编辑

- 格式化 `opt + shift + f`

{{< /tab >}}
{{< tab "Windows" >}}

## 选择

- 选中所有同一个词 `ctrl + shift + L`

## 跳转

- 左右括号跳转 `ctrl + shift + \`
- 跳到定义 `ctrl + 鼠标左键单击`
- 返回光标上一个位置 `alt + ←`
